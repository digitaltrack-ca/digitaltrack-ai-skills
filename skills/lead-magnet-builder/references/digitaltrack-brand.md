# DigitalTrack Brand Reference

Primary source: https://github.com/digitaltrack-ca/DIGITALTRACK_BRAND

Refresh this file from `docs/brand/DIGITALTRACK_BRAND_AGENT.md` when the official brand repository changes.

## Identity

- Brand: `DigitalTrack`
- Client-facing signature/business reference: `DigitalTrack.co`
- Positioning: practical, bilingual marketing for local service businesses so they can be found online, trusted, and chosen

## Visual system

- Primary cyan: `#40D6E4`
- Dark gray: `#606165`
- Black: `#000000`
- White: `#FFFFFF`
- Typeface: Montserrat; Arial fallback

Logo rules:

- do not distort, rotate, rearrange, or recolor the lockup
- preserve whitespace
- minimum width: 90 px digital / 1.25 in print
- use full color on light backgrounds
- use white on cyan or dark backgrounds

## Writing

Use direct language, practical outcomes, local-business examples, education, simple bilingual structure, credibility, and trust.

Avoid hype, unsupported guarantees, abstract mission language, and unnecessary enterprise jargon.
